Modular status bar for dwm written in c.


### Usage
    
    make

    sudo make install

After that you can put dwmblocks in your xinitrc or other startup script to have it start with dwm.

### Running dwmblocks

    
    dwmblocks



### Add scripts 


    mkdir -p ~/.local/bin
    cd ~/.local/bin

    chmod 755 battery volume percentage brightness
